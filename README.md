# Gothic Lock Calculator

Консольный калькулятор взлома замков для **Gothic 1 Remake**. Программа принимает описание замка (связи между пластинами и стартовые позиции штифтов) и находит **кратчайшую безопасную** последовательность ходов через BFS — без ударов о стены (позиции 1 и 7).

## Сборка

```bash
make
make test
```

Бинарник: `gothic-lock`

## Использование

```bash
gothic-lock tests/fixtures/tower_chest.txt
gothic-lock - < my_lock.txt
gothic-lock --template
gothic-lock --help
```

## Формат ввода

```
Name: Second chest in the tower
Rules:
1: 3r, 6l
2: -
3: 1r, 4l, 6r
4: 2r, 5r, 6l
5: -
6: 3l
Start:
[5, 3, 6, 7, 2, 7]
```

- **Rules** — по одной строке на пластину. При нажатии **D** (пластина вправо) записываете, какие *другие* пластины сдвинулись: `Nr` = вправо, `Nl` = влево. `-` = никто не двигается.
- **Start** — текущие позиции штифтов (1–7 слева направо). Цель — все на **4**.
- Поддерживается **2–8 пластин**.

## Как снять правила в игре

1. Выберите пластину и нажмите **D** (сдвиг вправо).
2. Посмотрите, какие *другие* пластины сдвинулись и куда.
3. Запишите связи в формате `Nr` / `Nl`.
4. Если пластина упёрлась в стену (позиция 1 или 7), сначала сдвиньте её к центру — иначе часть связей не видна.

## Управление в игре

| Клавиша | Действие |
|---------|----------|
| **D** | Пластина вправо |
| **A** | Пластина влево |
| **W** / **S** | Переключение между пластинами |

## Пример вывода

```
Second chest in the tower
Решение (52 шагов):
  1 пластина 1 раз вправо
  2 пластина 4 раза вправо
  ...
```

Подряд идущие одинаковые ходы сжимаются («2 пластина 4 раза вправо» = четыре раза D на пластине 2).
